## APPLY CONFIG

```
kubectl apply -f  https://gitlab.com/volnenko/kuber/-/raw/main/time-server.yml
```

## DELETE CONFIG
```
kubectl delete -f  https://gitlab.com/volnenko/kuber/-/raw/main/time-server.yml
```
